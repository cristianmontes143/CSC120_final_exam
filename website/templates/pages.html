Linear Regression

Linear regression is a supervised learning algorithm used to predict continuous output variables.

Linear Regression Code Documentation

The Linear Regression Model

The Linear regression model is used to create a predictor on Median house price based on 30yr Interest rate. The code works by training the model on the data of median house rate and 30yr interest house rate establishing a relationship between the variables. But the Linear regression model only assumes that there is a linear relationship between the variables.

Logistic Regression

Logistic regression is a supervised learning algorithm used to predict binary or categorical output variables based on input variables.

Logistic Regression Code Documentation

The Logistic Regression

The Logical regression model is used to create a binary predictor on the possibility of heart disease it is based on heart diseases dataset the contains 5000 unique data. The model trains the data by iteratively updating the model's coefficients using a technique called maximum likelihood estimation. It uses a logistic function to map the linear combination of the input variables to a probability, which is then used to classify the data into different classes. Its main feature is the classification of the inputs of whether it has the possibility of having heart diseases or not.

Usage Instructions

To properly use the code you must enter the neccesary inputs on the UI or follow the colab link for the full code

  1. Provide the required inputs by setting the values for the following fields:
    • Age: Enter the age of the patient using the age_field input.
    • Sex: Select the sex of the patient (Male or Female) using the sex_field input.
    • Chest Pain Type: Choose the type of chest pain from the options provided using the cp_field input.
    • Resting Blood Pressure: Enter the resting blood pressure of the patient in mm Hg using the trtbps_field input.
    • Cholesterol: Enter the cholesterol level of the patient in mg/dl using the chol_field input.
    • Fasting Blood Sugar: Select the fasting blood sugar level of the patient (<120 mg/dl or >120 mg/dl) using the fbs_field input.
    • Resting ECG: Choose the restingographic results from the options provided using the restecg_field input.
    • Maximum Heart Rate Achieved: Enter the maximum heart rate achieved by the patient using the thalachh_field input.
    • Exercise Induced Angina: Select whether the patient experienced exercise-induced angina (No or Yes) using the exng_field input.
    • ST Depression Induced by Exercise: Enter the ST depression induced by exercise value using the oldpeak_field input.
    • Slope of the Peak Exercise ST Segment: Choose the slope of the peak exercise ST segment from the options provided using the slp_field input.
    • Number of Major Vessels Colored by Flourosopy: Select the number of major vessels (0-3) colored by flourosopy using the caa_field input.
    • Thalassemia: Choose the thalassemia type from the options provided using the thall_field input.
  2. After providing the inputs, run the code and wait for the prediction to be generated. The possible outputs are as follows:
    • If the output is 0, the model predicts that there is a risk of heart disease. This will be displayed as "The model predicts that there is a risk of heart disease."
    • If the output is not 0, the model predicts that there is no risk of heart disease. This will be displayed as "The model predicts that there is no risk of heart disease."

    • Scikit-learn is a powerful machine learning library in Python. It provides a wide range of algorithms and tools for tasks such as classification, regression, clustering, and dimensionality reduction. Scikit-learn is used in the code for model evaluation and performance metrics.
  5. Matplotlib
    • Matplotlib is used to visualize data, explore patterns, and communicate results in a visual format.
  6. Seaborn
    • Seaborn is used to simplify the process of creating complex visualizations and offers additional statistical features for exploring and analyzing data.
  7. Math
    • The math module is used in logistic regression for calculations involving mathematical formulas and computations.

About Dataset

Heart Disease Diagnosis Datasets: Detailed Information This collection includes four datasets related to heart disease diagnosis, sourced from esteemed medical institutions: Cleveland Clinic Foundation, Hungarian Instiology, V.A. Medical Center in Long Beach, CA, and University Hospital in Zurich, Switzerland. Each database shares a common instance format, with 76 raw attributes, although only a subset of 14 attributes has been extensively used in past experiments. The "goal" field within these databases indicates the presence of heart disease in patients, with integer values ranging from 0 (no presence) to 4. Notably, the Cleveland database has been widely utilized by machine learning researchers in heart disease studies. To respect privacy, the names and social security numbers of patients have been replaced with dummy values. In addition to the processed Cleveland database, the directory also contains the four unprocessed files. Number of instances: - Cleveland: 303 - Hungarian: 294 - Switzerland: 123 - Long Beach VA: 200 Attribute Information: The dataset comprises 76 attributes, with published experiments typically focusing on a subset of 14 attributes. These attributes include patient demographics, medical measurements, and history. Complete attribute documentation is available for in-depth analysis. The datasets provide valuable insights into factors contributing to heart disease presence, allowing researchers and healthcare professionals to explore patterns, conduct experiments, and develop accurate diagnostic models. Class Distribution: The class distribution varies across the databases, with different frequencies for each category of heart disease diagnosis. Please refer to the specific dataset for detailed class distribution information. Please note that the dataset may contain missing attribute values, which are denoted with the value -9.0.
